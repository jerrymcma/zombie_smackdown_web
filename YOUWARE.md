# Zombie Puncher Game

A fast-paced arcade game where you punch zombies to earn points!

## Features
- **Whac-A-Mole Style Gameplay**: Zombies pop up at random locations.
- **Custom Cursor**: Use a giant fist to smash zombies.
- **Score System**: Earn points for every hit.
- **Timer**: Race against the clock (60 seconds).
- **Animations**: Smooth pop-up and punch animations using Framer Motion.

## Tech Stack
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Assets
- Zombie Character: Pixabay
- Background: Pixabay (Haunted House)
- Fist Cursor: Pixabay

## How to Play
1. Click "START RAMPAGE" to begin.
2. Move your mouse to aim the fist.
3. Click on zombies before they disappear.
4. Try to get the highest score before time runs out!
